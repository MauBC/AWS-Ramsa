# AWS Ramsa Platform

Proyecto fullstack compuesto por:

- Frontend → React (Vite)
- Backend → Node.js (Serverless Framework)

---

## 📦 Requisitos

- Node.js instalado
- npm instalado
- Serverless Framework instalado (para backend)

Instalar serverless globalmente si no lo tienes:

npm install -g serverless

---

## 🚀 Correr el Frontend

Entrar a la carpeta del frontend:

cd FrontEnd
npm install
npm run dev

Luego abrir en el navegador:

http://localhost:5173

---

## ⚙️ Correr el Backend

Entrar a la carpeta del backend:

cd BackEnd/MarcelaAWS
npm install

### Ejecutar localmente (si usas serverless offline)

serverless offline

### Desplegar en AWS

serverless deploy

---

## 📁 Estructura del proyecto

AWS-Ramsa
│
├── FrontEnd
│   └── React App
│
└── BackEnd
    └── Serverless Node API
